# PRODIGY_WD_05
A responsive Weather App built using HTML, CSS, and JavaScript that provides real-time weather information by fetching data from a weather API. Users can check the current weather using their device's location or by searching for a city name.
📌 Features
🌍 Get weather using the current location
🔍 Search weather by city name
🌡️ Display current temperature
💧 Show humidity
🌬️ Display wind speed
☁️ Show weather conditions and icons
📱 Fully responsive design
⚡ Real-time weather updates
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Weather API
